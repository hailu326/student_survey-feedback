# student_survey&feedback
A responsive web application for managing student surveys, featuring secure login, database integration, and a stakeholder dashboard

A responsive web application designed to manage student surveys efficiently. This system allows educational institutions to gather valuable feedback from students through a secure and user-friendly interface.

## 🚀 Key Features
- **Secure Authentication:** Implementation of a secure login system for students and administrators.
- **Survey Management:** Easy creation and submission of feedback forms.
- **Interactive Dashboard:** A dedicated dashboard for stakeholders to visualize and analyze survey data.
- **Responsive Design:** Optimized for both desktop and mobile devices.

## 🛠 Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Project Management:** Managed and tracked using **Jira** (Sprints & Backlog).
- **Version Control:** GitHub for collaborative development.
